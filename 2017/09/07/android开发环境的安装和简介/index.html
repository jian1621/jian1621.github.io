Android Studio的安装和测试

发表于 2017-09-07   |   分类于 学习   |  

android studio安装

  • 安装jdk和jre,设置环境变量，cmd 通过java -version 指令来验证是否安装成功。
  • 下载android studio 集成版本，设置好sdk和avd的目录，进行安装。
  • 在不能够实现avd的时候，可以采用genymotion模拟器，或采用真机来进行模拟。建议采用真机,因为这样比较快,如果采用真机的话,就需要进行安装驱动,再调用adb进行安装apk.
    参考文献：http://ask.android-studio.org/?/question/789

开发环境介绍

  • 参考《第一行代码 第2版》–郭霖第一章，认识各个部分的含义，主要是app和gradle部分，能够找到常用切换的位置，能够配置项目的基本参数，会使用日志工具log输出我们想看到的数据，
    这个和eclipse里面的开发很像，转换过来也是极为方便的。
  • 不同点在于gradle的使用，需要特意注意这方面知识的学习。
